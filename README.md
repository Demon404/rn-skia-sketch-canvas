# rn-skia-sketch-canvas

A React Native component for drawing by touching using Skia renderer

## Installation

```sh
npm install rn-skia-sketch-canvas
```

## Usage

```js
import { multiply } from "rn-skia-sketch-canvas";

// ...

const result = await multiply(3, 7);
```
